# Weather API Project
XJCO3011 Coursework 1

## Project Overview
A simple and complete Django RESTful API for managing cities and weather information.

## Features
- City Management (CRUD)
- Weather Records (CRUD)
- Filter weather by city
- Beautiful frontend dashboard
- Swagger API documentation
- Django admin panel
- Clean code & full English comments

## Tech Stack
- Django 5
- Django REST Framework
- SQLite
- HTML/CSS/JavaScript

## API Endpoints
- /api/cities/ - City management
- /api/weathers/ - Weather management
- /swagger/ - API documentation
- / - Frontend dashboard

## How to Run
1. pip install -r requirements.txt
2. python manage.py migrate
3. python manage.py runserver
